# MedMinder - Medication Tracker

A calming, accessible medication tracking web app designed for elderly users. Track daily medications with an easy-to-use interface featuring clock-based organization and simple progress tracking.

![MedMinder](https://img.shields.io/badge/MedMinder-Medication%20Tracker-4A9B9B?style=for-the-badge)

## Features

- **Clock-Based Organization** - Medications grouped by time of day (Morning, Afternoon, Evening, Night)
- **List View** - All medications sorted chronologically
- **Easy Tracking** - One-tap "Mark as Taken" buttons
- **Progress Dashboard** - Visual progress bar showing daily completion
- **Accessible Design** - Large fonts, high contrast, simple navigation
- **No Account Required** - Data stored locally in your browser
- **Works Offline** - Save as app or use offline

## Accessibility Features

- Minimum 18px font size for readability
- High contrast color scheme (WCAG AA compliant)
- Large touch targets (48px minimum)
- Keyboard navigation support
- Screen reader friendly with ARIA labels
- Calming color palette (soft teal and cream)

## Getting Started

### Option 1: Open Directly
Simply open `index.html` in any modern web browser.

### Option 2: Use as Web App
1. Open `index.html` in your browser
2. On mobile: Add to Home Screen
3. On desktop: Bookmark or use browser extension to "Install"

### Option 3: Host on GitHub Pages
1. Fork or clone this repository
2. Go to repository Settings > Pages
3. Select "Deploy from branch" and choose `main`
4. Your app will be live at `https://yourusername.github.io/medminder`

## Usage

1. **Add Medication**: Click the blue "+" button at the bottom right
2. **Enter Details**: Name, dosage, frequency, and time(s)
3. **Mark as Taken**: Click the "Mark Taken" button when you take a medication
4. **Track Progress**: View your daily progress at the top of the app
5. **Switch Views**: Use tabs to toggle between "By Time" and "By List" views
6. **Edit/Delete**: Use the Edit and Delete buttons on each medication card

## Technology

- Pure HTML, CSS, and JavaScript

## Privacy

All data is stored locally in your browser's LocalStorage. No data is sent to any server. Your medication information never leaves your device.
