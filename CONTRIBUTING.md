# Contributing to MedMinder

Thank you for your interest in contributing!

## How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/kalrakrish777-ctrl/medminder.git
   cd medminder
   ```

2. **Open in browser**
   Simply open `index.html` in any modern web browser. No build step required!

3. **Development**
   - Edit `index.html` directly
   - Refresh the browser to see changes
   - Use browser devtools (F12) for debugging

## Submitting Changes

1. Create a new branch:
   ```bash
   git checkout -b my-feature
   ```

2. Make your changes and commit:
   ```bash
   git add .
   git commit -m "Description of changes"
   ```

3. Push and create a Pull Request:
   ```bash
   git push origin my-feature
   ```

## Code Style

- Keep JavaScript simple (this is a vanilla React + Babel project)
- Use existing CSS variables for colors
- Test on mobile and desktop viewports

## Issues

- Use the issue templates for bug reports and feature requests
- Be descriptive and include steps to reproduce for bugs
